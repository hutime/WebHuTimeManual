グラフのレコードセット


１．概要

グラフには、グラフ用のレコード（HuTime.ChartRecord）、および、それらを収容するグラフ用のレコードセット（HuTime.ChartRecordset）が用いられる。値に用いるデータ項目やプロットや線の書式など、グラフを表示するために必要な情報は、レコードセットにを設定する。

２．レコードセットの生成

グラフ用のレコードセット（HuTime.ChartRecordset）は、以下のコンストラクタにより生成する。t値が時間データで暦に基づく表現である場合は、HuTime.CalendarChartRecordset を用いる。

コンストラクタ

HuTime.ChartRecordset(source, tBeginItem, tEndItem, valueItem [, plotStyle [, lineStyle ]])

HuTime.CalendarChartRecordset(source, tBeginItem, tEndItem, valueItem, calendarId [, plotStyle [, lineStyle ]])

source
型：String | HuTime.StreamReaderBase
レコードセットを読み込むファイルのパスまたはURL、もしくは、ストリームリーダ。
tBeginItem
型：String｜Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
レコードのt値の始点を示すデータの項目名（列名）または列番号。
tEndItem
型：String｜Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
レコードのt値の終点を示すデータの項目名（列名）または列番号。null や無効な値が与えられた場合は、tBeginItem と同じ値が設定される。
valueItem
型：String｜Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
グラフの値を示すデータの項目名（列名）または列番号。複数のデータ系列を持つグラフを作成する場合は、レコードセットオブジェクトを生成した後、selectValueItem メソッドでグラフの値となる項目を選択する。
calendarId
（HuTime.CalendarChartRecordsetのみ）
型：String
t値に用いる時間データの暦ID。
plotStyle
型：HuTime.FigureStyle | Function
レコードセット標準のグラフのプロットの書式。省略された場合は既定値が適用される。書式を出力する関数を指定する方法は、７．プロットの表示に関する設定を参照。
lineStyle
型：HuTime.FigureStyle | Function
レコードセット標準のプロット間の線の書式。省略された場合は既定値が適用される。書式を出力する関数を指定する方法は、８．プロット間の線の表示に関する設定を参照。

３．データの読み込み

コンストラクタでsource 引数を設定するか生成されたオブジェクトのreader プロパティを設定した後、loadRecordset メソッドにより、データの読み込みが実行する。

メソッド

loadRecordset()

データの読み込みを開始する。

返値
なし

プロパティ

reader
型：HuTime.StreamReaderBase
読み込みに用いるストリームリーダ。読み込み状態は、reader.loadState プロパティにより確認できる。
source
型：String
読み込み元のURL。reader.sourceを参照している。

４．レコードの追加と削除

グラフには、レコードとしてHuTime.ChartRecordを用いる。それぞれのレコードは、レコードセットオブジェクトの records 配列に収容される。レコードの追加や削除は、appendRecord、removeRecordなどのメソッドを通じて行う。

records
型：Array
読み取り専用
レコードセットが持つレコードを収容する。読み取り専用であるため、recordを別の配列に置き換えるなどの操作はできない。push等の操作は可能でが、他の処理との整合性を保つため、レコードの追加・削除は、appendRecord、removeRecordなどのメソッドを通じて行うべきである。

appendRecord(record)

レコードをレコードセットに追加する。

返値
型：HuTime.ChartRecord
追加されたレコードオブジェクト。追加に失敗した場合は、null。
record
型：HuTime.ChartRecord
追加するレコードオブジェクト。

removeRecord(record)

レコードをレコードセットから削除する。

返値
型：Number
削除したレコードの数。
record
型：HuTime.ChartRecord | Number | Function
削除するレコードオブジェクトを以下の方法で指定する。
  • 削除するレコードをHuTime.ChartRecordオブジェクトとして指定。
  • 削除するレコードをrecordsの要素番号として指定。
  • 削除するレコードを判定する関数を指定。レコードセットが持つ全てのレコードに指定された関数が適用され、削除の可否が判定される。

    func(record)

    返値
    型：Boolean
    trueを返した場合、引数recordのレコードは削除される。
    record
    型：HuTime.ChartRecord
    削除の可否を判定するレコード。removeRecordの処理内でレコードセット内のレコードが順番に渡される。

５．レコード項目（データ系列）に関する設定

レコードのどの項目をグラフの値とするかは、selectValueItem および、deselectValueItem メソッドで設定する。これらのメソッドを用いることにより、１つのレコードセットから複数のデータ系列を持つグラフを作成することができる。

selectValueItem(itemName [, plotStyle [, lineStyle [, order]]])

グラフの値とするデータ項目を選択し、その項目の書式を設定する。既存の設定（itemNameが一致）がある場合は、書式設定や表示順が上書きされる（書式等を省略した場合は、既定値で上書き）。

返値
型：Boolean
選択が成功した場合はtrue、失敗した場合はfalse。
itemName
型：String | Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
グラフの値として使うデータ項目名（列名）または列番号。
plotStyle
型：HuTime.FigureStyle | Function
選択したデータ項目のプロットの書式。省略された場合は既定値が適用される。書式を出力する関数を指定する方法は、７．プロットの表示に関する設定を参照。
lineStyle
型：HuTime.FigureStyle | Function
選択したデータ項目のグラフの線（プロット間の）書式。省略された場合は既定値が適用される。書式を出力する関数を指定する方法は、８．プロット間の線の表示に関する設定を参照。
order
型：Number
設定するデータ項目の表示順。棒グラフによる積み重ねグラフでは、値が大きいほど上に表示される。省略された場合は既定値（NaN）が適用され、積み重ねグラフでは一番下に表示される。同値の場合の表示順は不定。

deselectValueItem(itemName)

グラフの値として選択されているデータ項目を非選択とする。当該項目の書式等の設定は削除される。

返値
型：Boolean
選択が解除された場合はtrue、該当項目が無かった場合や解除に失敗した場合はfalse。
itemName
型：String | Number
グラフの値として使うデータ項目名（列名）または列番号。
disableSortRecords
型：Boolean
既定値：false
trueに設定すると、t値に基づいたレコードのソートが抑止される。抑止しない場合は、元データの順序に従ってデータが表示される。

６．レコードセット全体の表示に関する設定

複数のレコード項目（データ系列）を表示する場合に、個々のレコード項目単位でなく、レコードセット全体でデータの表示・非表示を設定する。表示するレコード項目が１つである場合は、showPlot プロパティ、showLine プロパティを使っても表示・非表示の制御は可能である。レイヤのshowLineプロパティ、または、showPlotプロパティで非表示の設定がなされている場合は、レコードセットの設定は無視され、値にかかわらず、非表示となる。

showRecordset
型：Boolean
既定値：true
レコードセット全体のプロットとプロット間の線の表示・非表示を設定する。trueで表示。
showRecordsetPlot
型：Boolean
既定値：true
レコードセット全体のプロットの表示・非表示を設定する。trueで表示。
showRecordsetLine
型：Boolean
既定値：true
レコードセット全体のプロット間の線の表示・非表示を設定する。trueで表示。

７．プロットの表示に関する設定

グラフ上のプロット（棒グラフの場合は棒）について、表示の可否や書式などを設定する。表示の可否、書式、シンボルの種類、プロットの大きさ、プロットの回転角は、固定値だけでなく関数を設定することもできるため、それぞれの設定値を、レコードの内容に合わせて変化させることも可能である（棒グラフでは、表示の可否と書式のみ）。

プロットの表示・非表示

プロット（棒グラフの場合は棒）の表示・非表示を設定する。関数を設定することにより、何らかの条件を満たすレコードのプロットを非表示とする操作が可能になる（例えば、別のデータ項目でエラーとマークされたレコードを非表示にする）。

showPlot
型：Boolean | Function
既定値：true
レコードセットのプロットの表示・非表示の標準値。グラフのレコード項目（データ系列）が１つの場合やすべてのグラフのレコード項目を同様に表示・非表示に設定する場合は、setItemShowPlot メソッドの代わりに、このプロパティでプロットの表示・非表示を設定しても問題ない。nullを指定すると、既定値（true）に戻る。無効な値を設定した場合は無視される。
  • 真偽値を指定。trueで表示。
  • 表示・非表示を判定する以下の形式の関数を指定。

    func(itemName, record)

    返値
    型：Boolean
    trueを返した場合、そのプロットは表示される。
    itemName
    型：String
    表示の可否を判断するプロットのデータ項目名（列名）または列番号が渡される。
    record
    型：HuTime.ChartRecord
    表示の可否を判断するプロットのレコードが渡される。

setItemShowPlot(itemName, show)

データ項目ごとのプロットの表示・非表示を設定する。

返値
なし
itemName
型：String | Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
プロットの表示・非表示を設定するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（showPlot プロパティ）に割り当てられている。
show
型：Boolean | Function
プロットの表示・非表示の設定値。nullを指定すると、設定が削除され、標準値（showPlot プロパティ）が適用されるようになる。無効な値を入力した場合は、無視される。
  • 真偽値を指定。trueで表示。
  • 表示・非表示を判定する関数。関数の形式は、showPlot と同様。

getItemShowPlot(itemName)

データ項目ごとのプロットの表示・非表示を確認する。データ項目ごとの指定が無い場合や、itemName の指定が正しくない場合は、レコードセットの標準値（showPlot）の内容が返される。

返値
型：Boolean | Function
指定されたデータ項目（itemName）のプロットの表示・非表示の設定値または判定関数。
itemName
型：String | Number
プロットの表示・非表示を取得するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（showPlot プロパティ）に割り当てられており、showPlotの設定内容が返される。
hidePlotNonRRange
型：Boolean
既定値：false
確実期間のないレコードのプロットの非表示・表示。true で非表示。
hidePlotTotalPRangeOnly
型：Boolean
既定値：false
全可能期間のみのレコードのプロットを非表示・表示。true で非表示。

プロットの書式

プロット（棒グラフの場合は棒）の書式（色など）を設定する。関数を設定することにより、条件ごとに異なる書式のプロットを表示することができる。

plotStyle
型：HuTime.FigureStyle | Function
既定値：HuTime.FigureStyle("#FF7F50")
プロットの書式のレコードセットとしての標準値。グラフのレコード項目（データ系列）が１つの場合やすべてのグラフのレコード項目のプロットを同じ書式に設定する場合は、setItemPlotStyle メソッドの代わりに、このプロパティでプロットの書式を設定しても問題ない。nullを指定すると、既定値に戻る。
  • 書式を HuTimeFigureStyle の固定値で指定。
  • 書式を出力する以下の形式の関数を指定。

    func(itemName, record)

    返値
    型：HuTime.FigureStyle
    返値の書式がプロットに適用される。
    itemName
    型：String | Number
    書式を適用するプロットのデータ項目名（列名）または列番号が渡される。
    record
    型：HuTime.ChartRecord
    書式を適用するプロットのレコードが渡される。

setItemPlotStyle(itemName, style)

データ項目ごとのプロットの書式を設定する。

返値
なし
itemName
型：String | Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
プロットの書式を設定するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（plotStyle プロパティ）に割り当てられている。
style
型：HuTime.FigureStyle | Function
プロットの書式の設定値。nullを指定すると、設定が削除され、標準値（plotStyle プロパティ）が適用されるようになる。無効な値を入力した場合は、無視される。
  • HuTime.FigureStyleで書式を指定。
  • 書式を返す関数。関数の形式は、plotStyle と同様。

getItemPlotStyle(itemName)

データ項目ごとのプロットの書式を取得する。データ項目ごとの指定が無い場合や、itemName の指定が正しくない場合は、レコードセットの標準値（plotStyle）の内容が返される。

返値
型：HuTime.FigureStyle | Function
指定されたデータ項目（itemName）のプロットの書式の設定値または判定関数。
itemName
型：String | Number
プロットの書式を取得するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（plotStyle プロパティ）に割り当てられている。

プロットのシンボル

プロット（棒グラフの場合は設定無効）のシンボルを設定する。関数を設定することにより、条件ごとに異なるシンボルのプロットを表示することが可能になる。

plotSymbol
型：Number | Function
制限：Number型では 0 ～ 3
既定値：0
プロットのシンボルのレコードセットとしての標準値。0:丸, 1:四角, 2:三角, 3:十字。グラフのレコード項目（データ系列）が１つの場合やすべてのグラフのレコード項目を同じシンボルに設定する場合は、setItemPlotSymbol メソッドの代わりに、このプロパティでプロットのシンボルを設定しても問題ない。nullを指定すると既定値（0:丸）に戻る。無効な値を指定した場合は無視される。
  • シンボルを固定値で指定。
  • シンボルを表す数値を出力する以下の形式の関数を指定。

    func(itemName, record)

    返値
    型：Number
    返値のシンボルを表す数値がプロットに適用される。
    itemName
    型：String | Number
    シンボルを設定するプロットのデータ項目名（列名）または列番号が渡される。
    record
    型：HuTime.ChartRecord
    シンボルを設定するプロットのレコードが渡される。

setItemPlotSymbol(itemName, symbol)

データ項目ごとのプロットのシンボルを設定する。

返値
なし
itemName
型：String | Number
制限：String 型では、空文字および空白のみは不可。Number 型では、NaN および無限大は不可。
プロットのシンボルを設定するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（plotSymbol プロパティ）に割り当てられている。
symbol
型：Number | Function
プロットのシンボルを表す数値。nullを指定すると、既定値（0：丸）が設定される。nullを指定すると、設定が削除され、標準値（plotSymbol プロパティ）が適用されるようになる。無効な値を入力した場合は、無視される。
  • シンボルの種類を表す数値。
  • シンボルの種類を表す数値を返す関数。関数の形式は、plotSymbl と同様。

データ項目ごとのプロットのシンボルを確認する。データ項目ごとの指定が無い場合や、itemName の指定が正しくない場合は、レコードセットの標準値（plotSymbol）の内容が返される。

返値
型：Number | Function
指定されたデータ項目（itemName）のプロットのシンボルを表す数値または判定関数。
itemName
型：String | Number
プロットのシンボルを取得するデータ項目名（列名）または列番号。なお、"default" はレコードセットの標準値（plotSymbol プロパティ）に割り当てられている。

プロットの大きさ

プロットの大きさを設定する。棒グラフの場合は、plotWidthType プロパティが 3（XY値固定）に設定された場合のみ、棒の幅を任意に設定できる。関数を設定することにより、他のデータ項目の値を使ってプロットの大きさを変える表現が可能になる。

plotWidth
型：Number | Function
既定値：10
プロットの大きさ（単位：px）のレコードセットとしての標準値。グラフのレコード項目（データ系列）が１つの場合やすべてのグラフのレコード項目のプロットを同じ大きさに設定する場合は、setItemPlotWidth メソッドの代わりに、このプロパティでプロットの大きさを設定しても問題ない。nullを指定すると、既定値（true）に戻る。無効な値を設定した場合は無視される。
  • 数値による指定。
  • プロットの大きさを出力する以下の形式の関数を指定。

    func(itemName, record)

    返値
    型：Number
    返された数値（単位：px）に従った大きさのプロットが表示される。
    itemName
    型：String | Number
    大きさを指定するプロットのデータ項目名（列名）または列番号が渡される。
    record
    型：HuTime.ChartRecord
    大きさを指定するプロットのレコードが渡される。

プロットのシンボルの回転角

プロットのシンボルの回転角を設定する（棒グラフの場合は無効）。四角のシンボルをひし形にする、十字のシンボルをXにする際などに利用する。関数による設定も可能である。

plotRotate
型：Number | Function
既定値：0
プロットのシンボルの回転角（単位：deg）のレコードセットとしての標準値。グラフのレコード項目（データ系列）が１つの場合やすべてのグラフのレコード項目のシンボルを同じ回転角に設定する場合は、setItemPlotRotate メソッドの代わりに、このプロパティでプロットのシンボルを設定しても問題ない。nullを指定すると、既定値（0）に戻る。無効な値を設定した場合は無視される。
  • 数値による指定。
  • プロットの回転角を出力する以下の形式の関数を指定。

    func(itemName, record)

    返値
    型：Number
    返された数値（単位：deg）に従って回転させたプロットのシンボルが表示される。
    itemName
    型：String | Number
    シンボルの回転角を指定するプロットのデータ項目名（列名）または列番号が渡される。
    record
    型：HuTime.ChartRecord
    シンボルの回転角を指定するプロットのレコードが渡される。

データ項目ごとのプロットのシンボルの回転角を設定する。

返値

範囲の表示処理をカスタマイズするする場合に設定する。既定の処理に戻すには、null を設定する。

func(tRange, v, style, layer, recordset, record, itemName, canvas)

返値
なし
tRange
型：HuTime.TRange
t値の範囲を描画するt値の範囲を示す HuTime.TRange オブジェクトが渡される。
v
型：Number
棒グラフの棒またはプロットのv値が渡される。
style
型：HuTime.FigureStyle
t値の範囲の書式が渡される。
layer
型：HuTime.LineChartLayer | HuTime.PlotChartLayer | HuTime.BarChartLayer
描画対象のレイヤオブジェクトが渡される。
recordset
型：HuTime.ChartRecordset
描画対象のレコードセットが渡される。
record
型：HuTime.ChartRecord
描画対象のレコードが渡される。
itemName
型：String
描画対象のデータ項目名が渡される。
canvas
型：HTMLCanvasElement
描画対象の HTML canvas が渡される。
